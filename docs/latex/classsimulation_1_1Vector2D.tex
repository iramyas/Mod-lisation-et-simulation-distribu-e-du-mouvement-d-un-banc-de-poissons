\doxysubsection{Description détaillée}
Représentation d\textquotesingle{}un vecteur en 2 dimension pour simuler des comportements physiques. 

Cette classe gère les positions, vitesses et accélérations dans un espace 2D. Elle fournit toutes les opérations mathématiques pour la simulation de boids.
